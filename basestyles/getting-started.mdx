---
title: "Getting Started"
---

<Steps>
  <Step title="Add Base Styles to your project" titleSize="h2" >
    <a href="https://raw.githubusercontent.com/reallygoodsoftware/base-styles/refs/heads/main/base-styles.css" download="base-styles.css">Download base-styles.css</a>
  </Step>

  <Step title="Set Your Defaults" titleSize="h2">
    ```css base-styles-custom.css
      :root {
        --ui-font-family-headers: 'Inter', sans-serif;
        --ui-font-family: 'Inter', sans-serif;
        --ui-color-primary: #0070f3;
        --ui-shared-border-radius: 6px;
        --ui-shared-border-width: 2px;
      }
    ```
  </Step>

  <Step title="3. Use the classes to compose your UI" titleSize="h2">
    ```html
      <button class="ui-button"> My Button </button>
    ```
  </Step>

  <Step title="4.(Optional) Add your own variants" titleSize="h2">
    If we don't have a variant you need, just add your own. See [customization](./customization.md) for more.

    ```css base-styles-custom.css
    .ui-button {
      & .--embossed {
        box-shadow: 0 0 0 2px var(--ui-color-primary);
      }
    }
    ```
  </Step>
</Steps>
